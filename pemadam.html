5.2 PEMADAM KEBAKARAN

Sistem Pencegahan dan Penanggulangan Api di Bangunan

Tujuan Sistem Pemadam Kebakaran

Sistem pemadam kebakaran dalam bangunan bertujuan untuk mendeteksi, mengendalikan, dan memadamkan api sejak dini guna mencegah penyebaran kebakaran serta menjaga keselamatan jiwa dan aset.

Komponen Utama Sistem

  • Sprinkler: Sistem penyemprot otomatis yang aktif saat suhu ruang mencapai ambang tertentu. Umumnya terhubung ke jaringan pipa air bertekanan tinggi.
  • Hidran: Titik air bertekanan tinggi yang dapat digunakan petugas atau penghuni gedung untuk memadamkan api secara manual menggunakan selang.
  • Pemadam Api Ringan (APAR): Alat portabel untuk memadamkan api skala kecil, tersedia di lokasi strategis seperti koridor dan area publik.
  • Instalasi Transportasi Vertikal (Lift): Dalam kondisi darurat, lift khusus kebakaran harus tersedia untuk evakuasi atau akses petugas.

Seluruh sistem ini harus dipasang sesuai standar keselamatan nasional dan diuji secara berkala untuk memastikan fungsinya saat kondisi darurat.
